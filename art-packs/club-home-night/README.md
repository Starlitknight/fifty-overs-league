# Club Home — After Hours Rain

A matched quiet-night homepage scene for desktop and mobile. Both compositions
show the same fictional club after play: wet paving, a warmly lit pavilion, the
town beyond the oval, one seated player and one distant groundsman.

## Runtime files

| Runtime | Repository path | Dimensions | Size |
| --- | --- | ---: | ---: |
| Desktop | `client/art/home/hgd-after-hours-rain.webp` | 1915 × 821 | 75 KB |
| Mobile | `client/art/home/hgm-after-hours-rain.webp` | 941 × 1672 | 90 KB |

The files are separate compositions, not a landscape image and a mechanical
portrait crop. Both are registered in the homepage's `night` art pool with the
caption `After hours · rain on the pavilion`.

The lossless PNG masters are retained outside GitHub to avoid repository bloat.
The generated art contains no interface, text, labels, logos or real clubs.
